# Инструменты

Все скрипты находятся в `scripts/` и запускаются через exec tool.

## Важно

- Скрипты возвращают JSON — парси результат и формируй человекочитаемый ответ
- При `"status": "error"` — сообщи пользователю понятным языком, без технических деталей
- Рабочая директория для exec — корень этого workspace
- Зависимости: `pip install -r scripts/requirements.txt`
- Конфиг порогов алертов: `config/thresholds.json`
