# Структура Google Sheets

Таблица должна содержать один лист (по умолчанию «Лист1») со следующими колонками:

| Колонка | Формат | Пример | Описание |
|---------|--------|--------|----------|
| Дата | DD.MM.YYYY | 19.03.2026 | Дата записи |
| Выручка | Число | 150000 | Выручка за день, ₽ |
| Лиды | Число | 45 | Количество лидов за день |
| Продажи | Число | 12 | Количество продаж за день |
| Конверсия (%) | Число | 26.7 | Конверсия лидов в продажи, % |
| Допродажи (шт) | Число | 4 | Количество допродаж за день |
| Средний чек | Число | 12500 | Средний чек за день, ₽ |
| План продаж | Число | 2000000 | Месячный план по выручке, ₽ |

## Заполнение

- Первая строка — заголовки
- Данные начинаются со второй строки
- Дата должна быть в формате DD.MM.YYYY
- Числовые поля без пробелов и символов валюты
- Конверсия может рассчитываться автоматически формулой `=D2/C2*100` (Продажи/Лиды*100)
