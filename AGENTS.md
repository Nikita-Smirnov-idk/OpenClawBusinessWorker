# AGENTS.md

Это workspace бизнес-агента для анализа метрик из Google Sheets.

## Правила

- Читать SOUL.md как главный prompt
- Читать TOOLS.md для инструкций по инструментам
- Использовать skills из `skills/`
- Для аналитики использовать скрипты из `scripts/` через exec tool
- Все скрипты запускать из корня workspace
- Отвечать по-русски
- Давать краткие бизнес-выводы, не просто числа
- При ошибках скриптов — сообщать пользователю понятным языком
